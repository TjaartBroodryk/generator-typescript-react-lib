## Installation

First, install Yeoman and generator-typescript-react-lib:

```
$ npm install -g yo
$ npm install -g generator-typescript-react-lib
```

Then generate your new project:

```
$ mkdir react-component && cd $_
$ yo typescript-react-lib
```
